# Playground

Tweak any option, hit **Run**, and see the result in real time. The two code panes show exactly what your action call and `<ModalTarget>` setup look like with the current configuration.

::Playground
::

## What you can change

- **Action** — `openModal`, `replaceModal`, or the `useModal` controller.
- **Modal props** — `title` and `level` passed to the demo dialog.
- **Open options** — `instantEnter` to skip the enter animation.
- **ModalTarget behavior** — every toggle from [`ModalBehaviorOptions`](/api/types#modalbehavioroptions): `enableInteractOutside`, `disableCloseOnInteractOutside`, `disableCloseOnInteractOverlay`, `disableLockBodyScroll`, `disableCloseOnEscape`.

The promise resolves when the user confirms (rendered in the demo dialog) and rejects when the modal closes — both outcomes show up in the **Last result** indicator.
